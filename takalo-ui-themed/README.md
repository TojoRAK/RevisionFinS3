# Takalo-takalo UI (Bootstrap 5) — Light/Dark

## Ouvrir
- Ouvre `client/index.html` dans ton navigateur.

## Thème
- Bouton soleil/lune dans le header (stocké via `localStorage` clé `tt_theme`).
- Le thème est appliqué via `data-theme="light|dark"` sur la balise `<html>`.

## Composants (à intégrer plus tard dans Flight)
- `components/header_client.html`
- `components/header_admin.html`
- `components/footer.html`
- `components/admin_sidebar.html`
- `components/modals.html`
- `components/toast_container.html`

## Pages
- Auth: `auth/login.html`, `auth/register.html`
- Client: `client/index.html`, `client/item_detail.html`, `client/my_items.html`, `client/offers.html`, `client/profile.html`
- Admin: `admin/login.html`, `admin/dashboard.html`, `admin/categories.html`
